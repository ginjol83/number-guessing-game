# Number Guessing Game 🎮

Esta es mi solución propuesta para el reto [Number Guessing Game](https://roadmap.sh/projects/number-guessing-game) de **roadmap.sh**. Es un juego clásico de adivinanzas basado en la línea de comandos (CLI) donde el jugador debe descubrir un número secreto generado aleatoriamente por la computadora antes de que se le agoten los intentos.

---

## 🚀 Características del Proyecto

### Requisitos Base Implementados:
* **Mensaje de bienvenida y reglas:** Explicación clara al iniciar la partida.
* **Generación aleatoria:** El sistema elige un número secreto entre **1 y 100**.
* **Niveles de dificultad:** * **Fácil:** 10 intentos.
    * **Medio:** 5 intentos.
    * **Difícil:** 3 intentos.
* **Retroalimentación en tiempo real:** El juego te indica si el número secreto es **mayor** o **menor** que tu suposición actual.
* **Contador de intentos:** Si ganas, te muestra exactamente cuántos intentos te tomó lograrlo.

### ⭐ Características Extra Incluidas:
* **Modo Multironda:** Al terminar una partida, puedes decidir si quieres volver a jugar o salir.
* **Temporizador:** Registra cuántos segundos tardaste en adivinar el número.
* **Puntuación Más Alta (High Score):** El sistema guarda el récord del menor número de intentos utilizados por nivel de dificultad durante la sesión.
